# Tesla News Dashboard (Tesla ニュースダッシュボード)

GoogleニュースRSSからTesla（テスラ）に関連する最新ニュースを自動収集し、カード形式で表示するダッシュボードアプリケーションです。
デザインはシンプルで読みやすいスタイル（[lowcarb.style](https://lowcarb.style/) 風）を採用しており、PC・スマートフォンの両方に対応しています。

## ⚡ 特徴

- **最新ニュースの自動収集**: Google News RSSを使用して、設定したキーワード（デフォルトは「Tesla ニュース」）に関連する記事をリアルタイムで取得します。
- **クリーンなデザイン**: 視認性の高いカード型レイアウトを採用。カード全体がクリック可能で、記事へのアクセスがスムーズです。
- **日本語UI**: アプリケーションのインターフェースはすべて日本語化されています。
- **互換性**: 複数のStreamlitバージョン（古いバージョンを含む）で動作するように設計されています。

## 🚀 インストール方法

### 前提条件
- Python 3.7以上
- pip

### 手順

1. リポジトリをクローンします:
   ```bash
   git clone https://github.com/lowcarbstyle/Tesla_News.git
   cd Tesla_News
   ```

2. 必要なライブラリをインストールします:
   ```bash
   pip install -r requirements.txt
   ```

## 🖥 使い方

以下のコマンドでアプリケーションを起動します:

```bash
streamlit run app.py
```

ブラウザが立ち上がり、ダッシュボードが表示されます（通常は `http://localhost:8501`）。
サイドバーから検索キーワードを変更することも可能です。
